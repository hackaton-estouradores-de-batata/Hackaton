# Slides

Aplicação separada para a apresentação do projeto.

## Rodar localmente

```bash
npm install
npm run dev
```

Abra `http://localhost:3001`.

## Observações

- Este módulo é isolado de `src/web`.
- A navegação dos slides é feita por clique na barra lateral e pelos botões de anterior/próximo.
- O conteúdo foi montado para diferenciar o que já foi entregue do que ainda é visão/roadmap.
