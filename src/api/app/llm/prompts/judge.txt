Considere apenas os dados fornecidos: caso estruturado, recomendação atual e resumo histórico.

Retorne apenas JSON válido no formato:
{
  "concorda": true,
  "observacao": null,
  "confianca_calibrada": 0.82
}

- `concorda` é booleano.
- `observacao` deve ser curta e objetiva; use `null` quando concordar.
- `confianca_calibrada` deve ficar entre 0 e 1.
